# portfolio2026

Personal portfolio of **Samijoe Hayek** — full-stack & blockchain developer and consultant, Dubai, UAE.

Built with [Astro](https://astro.build), GSAP and Lenis. Neo-brutalist design with an interactive illustrated hero (cursor-tracking eyes, blink, drifting tonal clouds and floating coins) composited in code from hand-cut layers.

## Develop

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # production build to dist/
```
